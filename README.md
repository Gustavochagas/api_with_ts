# Backend

RestAPI using nodeJS and typescript. Created database using migrates and seeds.
Technologies used:
  - NodeJS
  - Express
  - Knex
  - Sqlite3

### Running

Requires [Node.js](https://nodejs.org/) v12+ to run.

Install the dependencies and start the server.

```sh
$ npm install
$ npm run dev
$ running in http://localhost:3333
```
